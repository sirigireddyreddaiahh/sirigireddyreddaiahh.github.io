## 🚀 Installation & Deployment

1. **Clone the repository**  
   ```bash
   git clone https://github.com/sirigireddyreddaiahh/github.io.git
   ```

2. **Customize Content**  
   - Edit `index.html` to reflect your personal info, projects, and certifications.  
   - Add or remove images from the `assets/img/portfolio/` directory.

3. **Deploy to GitHub Pages**  
   - Rename the repository to: `github.io`
   - Push code to the `main` branch.
   - GitHub will automatically host it at `https://sirigireddyreddaiahh.github.io`

4. *(Optional)* Add your own [Google Analytics](https://analytics.google.com/) tracking ID inside the `<script>` tag if needed.
